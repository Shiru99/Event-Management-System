# Event-Management-System


## Installation

```
All Installions are added in corresponding folders 

1. Install Dart (Dart Folder)

2. Install Java 

3. Install Android Studio (Android Studio folder)

4. Install Flutter (Flutter folder)
```


# How to run

```
$ cd EventManager

$ flutter run
```
